## Requirements

* PHP >=7.4
* Docker, docker-compose

## Tests

```shell
$> ./vendor/bin/phpunit ./tests
```

## Build

```shell
$> docker-compose build
```

## Run

```shell
$> docker-compose up -d
```
Open http://localhost:8088/ip?name=Test
